# NEVA MOBILE

**NEVA MOBILE**, ikinci el telefon alım satımı yapan işletmeler için geliştirilen, tamamen offline çalışan profesyonel masaüstü yazılımıdır.

---

## Özellikler

- 📥 Telefon alış yönetimi
- 📤 Telefon satış yönetimi
- 💰 Kâr hesaplama
- 🏦 Kasa yönetimi
- 🛡️ Garanti takibi
- 🔑 Lisans sistemi
- 🔄 Otomatik güncelleme
- ⚡ Hızlı arama
- 📴 Tamamen offline çalışma
- 🗄️ Güvenli SQLite veritabanı

---

## Kimler İçin?

Bu yazılım;

- Telefon alım satımı yapan işletmeler
- GSM mağazaları
- İkinci el telefon satıcıları
- Telefon ticareti yapan bireysel satıcılar

için geliştirilmiştir.
